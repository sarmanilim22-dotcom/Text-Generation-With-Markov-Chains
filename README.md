# PRODIGY_GA_03 - Text Generation using Markov Chains

## Objective
Generate meaningful text using a Markov Chain model trained on a text dataset.

## Tools Used
- Python
- VS Code
- Markovify

## Dataset
A custom AI-related text dataset.

## How to Run

1. Install Markovify:

```bash
pip install markovify
```

2. Run:

```bash
python text_generation.py
```

## Output

The program generates new sentences based on the training data and saves them to `generated_text.txt`.
